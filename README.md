# Magnificent Diamond Website

A modern, responsive website for Magnificent Diamond - a leading manufacturer and supplier of lab-grown diamonds.

## Features

- Responsive design for all devices
- Hero image carousel
- Product showcase pages
- Contact form
- Modern UI with Tailwind CSS

## Deployment

This is a production build ready for deployment. The `index.html` is in the root folder with all assets in relative paths.

### GitHub Pages

1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select the branch and folder (root)
4. Your site will be live at `https://[username].github.io/[repository-name]`

### Other Hosting

Simply upload all files maintaining the folder structure:
- `index.html` (root)
- `static/` folder
- `images/` folder

All paths are relative and will work from any root URL.
